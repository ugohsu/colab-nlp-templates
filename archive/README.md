# 概要

- ライブラリの互換性がとれなくなった関数や使用しなくなった関数のうち、今後の拡張に参考になるファイルをここに保存します。

---

## 再利用・復元時の注意

ここにあるファイルは、元々 `libs/` ディレクトリ内で動作するように書かれています。
そのため、コード内には相対インポート（例: `from .preprocess import ...`）が含まれている場合があります。

これらを再利用・拡張して使う場合は、以下のいずれかの対応を行ってください。

1. **ファイルを `libs/` フォルダに戻して使う**（推奨）
   - 相対インポートが正しく機能するようになります。
2. **インポートパスを書き換える**
   - ファイルを移動せずに参考にする場合は、`from .preprocess` を `from libs.preprocess` などに修正する必要があります。
