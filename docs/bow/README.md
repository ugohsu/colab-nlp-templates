# Bag of Words（BoW）

本章では、**Bag of Words（BoW）** という考え方を軸に、  
日本語テキストを「単語の集合」として扱うための  
**最も基本的かつ王道的な分析手法群**を整理します。

BoW は、自然言語処理における

- 語頻度分析
- TF-IDF
- N-gram
- 共起分析
- WordCloud

といった多くの手法の**共通土台**となる考え方です。

本リポジトリでは、  
**形態素解析 → BoW → 各種分析・可視化**  
という流れを、Google Colaboratory 上で再現可能な形で提供します。

---

## Bag of Words（BoW）とは

Bag of Words（BoW）とは、文章を

> **「単語の集まり（袋）」**

として表現する、自然言語処理における最も基本的な考え方のひとつです。

BoW では、

- 単語の出現順序
- 文法構造
- 文脈的な意味関係

をすべて捨て、

- **どの単語が**
- **どれだけ出現したか**

だけに注目します。

## 位置づけ

Bag of Words は、

- 実装が簡単
- 結果が直感的に理解しやすい

一方で、語順や意味を捨てるという限界もあります。

本章では、BoW の強みと限界を理解し、  
後続の高度な NLP 手法への足がかりとすることを目的とします。
